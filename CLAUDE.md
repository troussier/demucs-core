# CLAUDE.md - demucs-core

## プロジェクト概要

Demucsの処理のみを担う共用コアライブラリ。UIには一切依存しない。
デミュ九郎 (mr-demu) およびその他のアプリから参照される。

---

## リポジトリ構成

```
demucs-core/
├── separator.py   # メインモジュール
├── README.md
├── CLAUDE.md      # このファイル
└── PROGRESS.md    # 進捗管理
```

---

## 開発ルール

- `separator.py` はUIに依存するコードを一切含めない
- 他アプリから呼び出しやすいようにAPIはシンプルに保つ
- モデル名・デバイス指定は呼び出し元から渡せるようにする
- エラーは例外として投げ、呼び出し元でハンドリングする
- コメントは日本語で書く

---

## 作業終了時のルール

作業を終了する際は必ず以下を行うこと：

1. `PROGRESS.md` を更新する
   - 今回やったこと
   - 次にやること
   - 未解決の問題があれば記載
2. 変更内容を適切なコミットメッセージでコミットする
